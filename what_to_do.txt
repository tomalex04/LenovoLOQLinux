Great plan! This is the most reliable way to get **ground truth** EC register maps. We'll use RWEverything to "snapshot" the EC RAM before and after Vantage makes changes, which tells us exactly which bytes control what.

Here's exactly what to do, step by step:

---

## 📋 Phase 1: Setup

1. Boot into **Windows 11**
2. Download & run **[RWEverything](http://rweverything.com/)** as **Administrator**
3. Open **Lenovo Vantage** and leave it running in the background

---

## 📸 Phase 2: EC RAM Baseline Snapshot

> This captures the "blank slate" before any changes.

1. In RWEverything, click the **"EC"** button at the top (Embedded Controller)
2. You'll see a grid of hex values — this is your EC RAM
3. **Take a screenshot** of the full EC RAM table
4. Also click **File → Save** and save it as `ec_baseline.txt`

---

## 🌀 Phase 3: Fan Curve Snapshots

> Do this for **each** fan speed level in Vantage.

In Vantage, go to **Power → Custom Mode → Fan Control**. Set the fan curve to **full manual** and:

### Step A — Set ALL points to **minimum speed (0% / lowest)**
- Save in Vantage
- **In RWEverything**: Click **"EC"** → **File → Save** → name it `ec_fan_min.txt`
- Take a screenshot

### Step B — Set ALL points to **maximum speed (100%)**
- Save in Vantage
- **In RWEverything**: **File → Save** → name it `ec_fan_max.txt`
- Take a screenshot

### Step C — Set ONLY Point 1 to 50%, leave all others at 0%
- Save in Vantage
- **In RWEverything**: **File → Save** → name it `ec_fan_pt1_50pct.txt`

### Step D — Set ONLY Point 2 to 50%, leave all others at 0%
- Save → name it `ec_fan_pt2_50pct.txt`

### Step E — Set ONLY Point 3 to 50%, leave all others at 0%
- Save → name it `ec_fan_pt3_50pct.txt`

> ⚠️ These 3 "single point" dumps are the most important — they let me locate exactly which bytes hold each individual curve point.

---

## ⚡ Phase 4: Wattage (Power Limit) Snapshots

In Vantage Custom Mode, go to **Power → CPU Power** (or wherever PL1/PL2/TDP sliders are):

### Step A — Set CPU TDP to **minimum** (lowest watt value)
- Save in Vantage
- **In RWEverything**: **File → Save** → name it `ec_watts_min.txt`

### Step B — Set CPU TDP to **maximum**
- Save in Vantage
- **In RWEverything**: **File → Save** → name it `ec_watts_max.txt`

### Step C — Set CPU TDP to exactly the **middle value**
- Save → name it `ec_watts_mid.txt`

---

## 🌡️ Phase 5: GPU Temperature Limit Snapshots

In Vantage, find the **GPU temperature limit** slider:

### Step A — Set GPU temp limit to **minimum**
- Save → `ec_gpu_temp_min.txt`

### Step B — Set GPU temp limit to **maximum**
- Save → `ec_gpu_temp_max.txt`
